IncludeExtension

The main driver behind developing the MooseDocs documentation system was to create a single source for all documentation and allow the same piece of text to be reused in multiple locations and in multiple formats. Therefore, it is possible to include markdown within markdown using the !include command. For example, Listing 1 provides the include command that is used below to capture text from the global extension.

Listing 1: An example !include command for retrieving markdown from another file.

!include docs/content/utilities/documentation/moose_markdown/extensions/global.md re=The\sglobal.*?applied\.

The following content is extracted from the global: !include docs/content/utilities/documentation/moose_markdown/extensions/global.md re=The\sglobal.*?applied.
